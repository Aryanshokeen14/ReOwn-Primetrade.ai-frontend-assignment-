Primetrade Full-Stack Assignment

Tech Stack       

Frontend: Next.js, Tailwind CSS     
Backend: Node.js, Express      
Database: MongoDB      
Authentication: JWT         
API Testing: Postman

Getting Started       

Backend       
cd backend       
npm install          
npm run dev  

Frontend       
cd frontend     
npm install    
npm run dev      


API Documentation        

A Postman collection is included at the project root.       
Steps:       
Import postman_collection.json into Postman      
Run the backend locally       
Test authentication and CRUD APIs      
JWT token is handled via Authorization headers
